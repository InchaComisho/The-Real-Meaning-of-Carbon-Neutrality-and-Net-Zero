## 概要

カーボンニュートラルとネットゼロは、一般に「実質的な排出ゼロ」と説明される。

この説明は部分的には正しい。

科学・政策の文脈では、ネットゼロとは、大気中へ放出される温室効果ガスの量と、大気中から除去される量が釣り合っている状態を指す。  
CO₂については、人為的CO₂排出量と人為的CO₂除去量が、一定期間において世界全体で均衡する状態として説明される。

しかし、この概念が会計、オフセット、クレジット、数値上の帳尻合わせだけに縮小されると、不完全になる。

```text
排出量 - オフセット = ゼロ
```

より深い意味は、以下である。

```text
排出削減
+
炭素吸収源の回復
+
炭素固定システムの回復
+
土壌・森林・海洋・湿地・微生物循環の回復
```

つまり、カーボンニュートラルやネットゼロは、人間社会が炭素を少なく排出することだけを意味するべきではない。

地球が炭素を吸収し、固定し、貯蔵し、再循環する能力を取り戻すことを意味するべきである。

## 1. カーボンニュートラルとは何か

カーボンニュートラルとは、一般に、排出された二酸化炭素や温室効果ガスの量が、同等の除去、吸収、またはオフセットによって釣り合っている状態を指す。

単純に言えば、以下である。

```text
排出量 - 除去量 = 実質ゼロ
```

または、

```text
排出されたCO₂
-
吸収または除去されたCO₂
=
実質ゼロ
```

カーボンニュートラルは、以下でよく使われる。

```text

```


これらは重要である。

しかし、問題は、カーボンニュートラルが単なる計算問題になったときに始まる。

カーボンニュートラルが貸借対照表のように扱われるなら、炭素を実際に制御する地球システムの回復を見落とす可能性がある。

## 2. ネットゼロとは何か

ネットゼロとは、大気中へ入る温室効果ガスと、大気中から除去される温室効果ガスが釣り合っている状態である。

CO₂については、世界全体でネットゼロCO₂排出を達成し、それを維持できれば、CO₂由来の追加的温暖化の進行を止める方向に近づくと考えられる。

単純化すると、以下である。

```text
大気へ排出される温室効果ガス
=
大気から除去される温室効果ガス
```

ネットゼロは、カーボンニュートラルよりも広い政策目標として使われることが多い。

それはしばしば以下を含む。

```text
CO₂
メタン
一酸化二窒素
フッ素系ガス
その他の温室効果ガス
```

しかし、ネットゼロは、実際の排出削減を遅らせる言い訳になってはならない。

信頼できるネットゼロ戦略は、まず可能な限り排出を削減し、それでも残る排出のみを、持続的で確実な除去によって補うべきである。

## 3. カーボンニュートラルとネットゼロの違い

カーボンニュートラルとネットゼロは、しばしば同じ意味で使われる。

しかし、常に完全に同じではない。

実務上の違いは、次のように整理できる。

```text

ネットゼロ：
CO₂だけでなく、複数の温室効果ガスを含む経済全体・世界全体の気候目標として使われやすい概念。
```

ただし、最も重要なのはラベルではない。

重要なのは、その戦略が大気と生態系において実際のバランスを作るかどうかである。

## 4. 会計上の問題


紙の上では中立に見えるかもしれない。

しかし、以下の問いが残る。

```text

生物多様性を回復しているか。
土壌炭素を回復しているか。
微生物循環を回復しているか。
水循環を守っているか。
```

これらを無視するなら、カーボンニュートラルは惑星回復戦略ではなく、数値上の主張にすぎなくなる。

## 5. ネットゼロはゼロ排出と同じではない

「ネットゼロ」という言葉は誤解を招きやすい。

ネットゼロは、必ずしもゼロ排出を意味しない。

残余排出を除去量で釣り合わせることを意味する。

```text
ゼロ排出：
排出そのものがない。

ネットゼロ：
一部の排出が残っても、除去によって釣り合う。
```

ここで重要なのは、除去がすべて同じではないという点である。

```text
一時的な除去もある。
不確実な除去もある。
反転リスクのある除去もある。
```

たとえば、以下が起こり得る。

```text
森林は燃える可能性がある。
土壌炭素は減少する可能性がある。
湿地は排水される可能性がある。
泥炭地は炭素を放出する可能性がある。
カーボンクレジットは除去量を過大評価する可能性がある。
オフセットは恒久的でない可能性がある。
```

化石炭素は、地下に何百万年も貯蔵されていた炭素である。

化石燃料を燃やすと、その炭素は大気・海洋・陸域の活発な炭素循環へ加わる。

これを一時的な生物貯蔵で完全に等価に扱うことは、常に適切とは限らない。

したがって、ネットゼロは慎重に設計されなければならない。

優先順位は以下である。

```text
まず深い排出削減。
残余排出には耐久性のある除去。
基盤として実際の生態系回復。
```

## 7. CO₂は原因であり、症状でもある

CO₂は温室効果ガスである。

大気中CO₂の蓄積は地球温暖化に寄与する。

これは否定すべきではない。

しかし、CO₂蓄積は、より深い惑星的失敗の症状でもある。

CO₂が増えたのは、人間が排出したからである。

しかし、CO₂が蓄積し続けるのは、地球の自然な吸収・固定システムが弱体化したからでもある。

したがって、CO₂は以下である。

```text
温暖化の直接的な物理要因
+
炭素循環不全の症状
```

CO₂を排出問題だけとして扱うと、解決策は狭くなる。

```text
排出を減らす。
オフセットを買う。
ネットゼロにする。
```

しかし、CO₂蓄積を炭素循環不全として理解すると、解決策は広がる。

```text
排出を減らす。
炭素吸収源を回復する。
土壌を再生する。
微生物循環を回復する。
森林を回復する。
湿地と泥炭地を保護する。
海洋生態系を回復する。
水循環を修復する。
自然な炭素固定システムを再構築する。
```

これが、カーボンニュートラルとネットゼロのより深い意味である。

土壌が劣化すれば、炭素と肥沃度を失う。

つまり、土壌再生は農業問題だけではない。

ネットゼロ問題である。  
カーボンニュートラル問題である。

土壌炭素を無視するカーボンニュートラル戦略は不完全である。

微生物は、カーボンニュートラルの宣伝ではほとんど語られない。

しかし、微生物は炭素固定と土壌炭素形成の中心である。


微生物ネクロマス、すなわち死んだ微生物細胞の残骸は、土壌有機炭素の重要な構成要素として注目されている。

つまり、微生物は単なる分解者ではない。

炭素処理者である。

炭素がすぐ大気へ戻るのか、土壌中で安定化するのかを左右している。

微生物生態系が崩壊すれば、土壌炭素システムは弱体化する。

土壌炭素が弱まれば、ネットゼロを物理的現実として達成することは難しくなる。

## 13. なぜオフセットだけでは解決できないのか

オフセットは限定的な役割を持ち得る。

しかし、オフセットは排出削減の代替にはならない。

主な問題は以下である。

```text
追加性：
その炭素除去は、プロジェクトがなくても起きていたのではないか。

永続性：
その炭素は十分長く貯蔵されるのか。

リーケージ：
一地域の保護が別地域の破壊を招いていないか。

二重計上：
同じ削減量を複数主体が主張していないか。

測定不確実性：
炭素効果は正確に測定されているか。

生態学的品質：
本物の生態系を回復しているのか、それとも炭素数値だけか。
```

これらが解決されなければ、オフセットは本当の惑星回復ではなく、気候対策の見かけを作るだけになり得る。

したがって、

```text
オフセットは一時的・補助的であるべきである。
排出削減が第一であるべきである。
炭素固定源回復は構造的対策であるべきである。
```

## 16. 惑星代謝回復としてのカーボンニュートラル

このモデルでは、以下のように見る。

```text
土壌は炭素処理器官である。
森林は炭素と水の調整器官である。
海洋は熱と炭素の調整器官である。
微生物は見えない代謝主体である。
湿地と泥炭地は長期炭素貯蔵器官である。
水循環は循環器系である。
```

真のカーボンニュートラルとは、惑星の代謝を回復することである。

---

## 17. 提案する因果モデル

本稿が提案する因果モデルは、以下である。

```text
人間活動
→ 化石燃料利用と土地利用変化
→ 森林破壊・土壌劣化・湿地消失・海洋ストレス
→ 微生物衰退と生態系の単純化
→ 炭素固定システムの弱体化
→ 炭素吸収源の弱体化
→ より少ない炭素しか吸収・固定・貯蔵・再循環されない
→ 大気中CO₂蓄積
→ 地球温暖化
→ 干ばつ・火災・洪水・海洋ストレス
→ 炭素吸収源へのさらなる損傷
→ 炭素固定システムのさらなる弱体化
```

このモデルは、なぜカーボンニュートラルが会計以上でなければならないかを示す。

気候システムはフィードバックループである。

自然炭素吸収源が弱まれば、大気中CO₂を安定させることは難しくなる。

したがって、解決策は以下でなければならない。

```text
排出削減
+
炭素固定システムの回復
+
自然吸収源の保護と再生
```

## 18. より完全な定義

より完全なカーボンニュートラルの定義は、以下である。

```text
カーボンニュートラルとは、排出量とオフセットの数値上の釣り合いだけではない。
人間の排出が深く削減され、残余排出が耐久的に除去され、さらに地球の自然な炭素固定システムが回復し、炭素が安定した生態循環の中で吸収・固定・貯蔵・再循環される状態である。
```

より完全なネットゼロの定義は、以下である。

```text
ネットゼロとは、温室効果ガスの排出と除去の釣り合いだけではない。
排出、除去、炭素吸収源、土壌炭素、海洋吸収、微生物循環、森林、湿地、水循環が一体となって、安定した地球システムとして機能する現実の惑星バランスの回復である。
```

## 20. 本モデルの独自貢献

カーボンニュートラルとネットゼロの重要性は広く認識されている。

排出削減の重要性も広く認識されている。

また、土壌、森林、海洋、湿地、微生物、炭素吸収源の重要性も既存科学で議論されている。

しかし、これらはしばしば別々に扱われる。

```text
カーボンニュートラル
ネットゼロ
カーボンオフセット
カーボンクレジット
土壌炭素
森林炭素
海洋炭素吸収源
微生物生態学
再生農業
生態系回復
```

本稿の貢献は、それらを一つの因果・戦略モデルに統合することである。

```text
カーボンニュートラルとネットゼロは、
地球の炭素固定システムを回復しない限り不完全である。
```

本稿は、欠けている戦略層として以下を提示する。

```text
カーボンニュートラルは、排出会計から
惑星炭素循環回復へ移行しなければならない。
```

## 21. 結論

カーボンニュートラルとネットゼロは必要である。

しかし、それらが数値目標だけに留まるなら不十分である。

社会は、紙の上でカーボンニュートラルを主張しながら、土壌が劣化し、森林が燃え、湿地が消え、海洋が弱まり、微生物炭素システムが崩壊する状態を続けることもできてしまう。

それは本当のカーボンニュートラルではない。

真のカーボンニュートラルとは、排出を減らし、地球が炭素を処理する能力を回復することである。

真のネットゼロとは、排出と除去を釣り合わせるだけではない。

炭素バランスを可能にするシステムを再構築することである。

気候戦略の未来は以下へ進むべきである。

```text
排出会計から
炭素固定源回復へ。
```

```text
ネットゼロ主張から
現実の地球システム回復へ。
```

```text
CO₂を減らすことだけから
地球がCO₂を吸収し、固定し、貯蔵し、再循環する能力を回復することへ。
```

カーボンニュートラルとネットゼロの本当の意味は、単に以下ではない。

```text
排出量をゼロにする。
```

それは以下である。

```text
炭素バランスを可能にする惑星システムを回復する。
```
